# 通用 cms 开发模板
[== 查看: 后端koa-api接口项目 ==](https://github.com/xiaominglin789/cms-common-vite-api)
- vue3.2+
- vite2.0+
- vue-router4.0+
- pinia2.0+
- elment-plus


## 下载
```bash
git clone https://github.com/xiaominglin789/cms-common-vite.git
```

## 安装
```bash
cd cms-common-vite
yarn
```


## 测试
```bash
yarn dev or yarn serve
```
